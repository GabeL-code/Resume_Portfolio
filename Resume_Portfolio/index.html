Encryption Decryption




Encryption Decryption project is a java project that uses a scanner or input form a console log to decrypt or encrypt a message.Through the console the user can input 3 variables corresponding to 3 commands “-mod”, “-key”, and “-data”. The “-mod” corresponds to the mode of either encrypting or decrypting. The “-key” is the character shift through the ascii decimal values. The “-data” is the messaged received. The video above is a mock demonstration of the project through the console log. In the video the “-data” - “Welcome to hyperskill” is encrypted and decrypted.




TicTacToe with self playing AI





This project is a single player tictacToe game that uses AI for the player to play against. The AI was developed using a backtracking algorithm. The program scans for a number input of 1 through 9 which corresponds to the postion number on the board. The CPU will then calculate the best postion to win agianst the player
